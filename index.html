Pasos para usar git

  1. Iniciamos git
  2. git init "para iniciar git por primera vez en el proyecto, no cada vez que trabajemos sobre nuestro proyecto."
  1. Pasos que usamos normalmente una vez ya iniciado el repositorio de git:
  2. git add . " el . es para guardar todos los archivos sin tener que hacerlo uno por uno"
  3. git commit -m "titulo de commit , donde comentamos el cambio que hicimos y que trabajamos"
  4. git status "es para verificar los estados o si tenemos algo para guardar en el commit "
  1. Ramas
  2. git branch "para listar las ramas del proyecto"
  3. git branch nombre-de-rama "para crear una rama en nuestro repositorio"
  4. git checkout nombre-de-rama "para movernos entre las ramas creadas"
  5. git merge nombre-de-rama "para fusionar los cambios de otras ramas a la principal, recordar estar en la rama master/main"
